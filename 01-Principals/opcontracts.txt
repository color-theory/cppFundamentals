Example of an operation contract

First-draft specifications:

sort(anArray, num)
// Sorts an array.
// Precondition: anArray is an array of num integers; num > 0.
// Postcondition: The integers in anArray are sorted.


Revised specifications:

sort(anArray, num)
// Sorts an array into ascending order.
// Precondition: anArray is an array of num integers
// and 1 <= num <= MAX_ARRAY, where MAX_ARRAY is a
// global constant that specifies the maximum size
// of anArray.
// Postcondition: anArray[0] <= anArray[1] <= ... <=
// anArray[num-1], num is unchanged
